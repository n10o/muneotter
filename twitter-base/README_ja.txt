#                                                                                 Overview
#                                                                                 ========
SAFlute の Example プロジェクト「Twitter」のベースプロジェクトです。

ベースプロジェクトは以下の役割があります。

  o Mavenの設定ファイル pom.xml の定義の共通化(parent)
  o プロジェクトの関するちょっとした環境設定ファイルやドキュメント(ERDなど)の置き場
  o プロジェクトを便利に利用するためのスクリプト(バッチファイル)などの置き場

この Example では、webプロジェクトのベースとなるプロジェクト(webベース)は作っていません。
もし、web独自のpomファイルの定義が多いようであれば、実際のアプリでは twitter-base-web
というようなプロジェクトを作っても良いでしょう。

※Example自体の説明や、実際のアプリで利用するための環境構築などは、
「twitter-base/README_ja.txt」をご覧ください。
