
# ========================================================================================
#                                                                                 Overview
#                                                                                 ========
SAFlute の Example プロジェクト「Twitter」の共通プロジェクトです。

共通プロジェクトは以下の役割があります。

  o DBFluteクライアントとモジュールを配置、自動生成されたクラスの置き場
  o Actionなどのスーパークラスや共通ロジックの置き場

ここでは、web依存した共通ロジックもそうでないロジックも区別せず管理されています。
もし、実際のアプリでそこを分離したい場合は、プロジェクトを分割したり、
パッケージを分けたり名前を変えたりして工夫すると良いでしょう。

※Example自体の説明や、実際のアプリで利用するための環境構築などは、
「twitter-nisetter/README_ja.txt」をご覧ください。
